# SimpleHEXViewer

Простой HEX просмотрщик / Simple HEX Viewer

## Описание / Description

### Русский
Репозиторий содержит готовый исполняемый файл (.exe) и папку с исходным кодом.

**Структура исходного кода:**
- `..\source\constants.h` - заголовочный файл с константами, используемыми в программе
- `..\source\DPI.h` - заголовочный файл с функциями для управления DPI
- `..\source\HelpersWinAPI.h` - заголовочный файл с вспомогательными функциями WinAPI
- `..\source\OpenHexFile.h` - заголовочный файл с функцией открытия файлов
- `..\source\SearchStructure.h` - заголовочный файл со структурой для окна поиска
- `..\source\WindowLocalStruct.h` - заголовочный файл со структурой главного окна
- `..\source\WindowsProc.h` - заголовочный файл с обработчиками оконных сообщений

### English
The repository contains a ready-to-run executable file (.exe) and a folder with source code.

**Source code structure:**
- `..\source\constants.h` - header file containing constants used in the program
- `..\source\DPI.h` - header file with functions for DPI management
- `..\source\HelpersWinAPI.h` - header file with auxiliary WinAPI functions
- `..\source\OpenHexFile.h` - header file with file opening function
- `..\source\SearchStructure.h` - header file with structure for search window
- `..\source\WindowLocalStruct.h` - header file with main window structure
- `..\source\WindowsProc.h` - header file with window message handlers
